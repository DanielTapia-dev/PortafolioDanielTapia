BACK-END

Este funciona mediante express y una base de datos relacional en PostgreSQL, se creo una API Rest para conectar el Front-End con el Back-End, para realizar la facturacion electronica el mismo tiene almacenada la firma electronica del cliente y se conecta directamente con el servidore de facturacion electronica del SRI y actualmente se encuentra en fase de produccion, vale recalcar que el sistema esta desarrollado con Javascript puro a diferencia del Front-End que utiliza TypeScript.

FACTURACION ELECTRONICA

Este es el modulo más importante del sistema ya que automatiza por completo el proceso de facturación, a continuacion se muestra la pantalla principal desde donde el cajero puede facturar facilmente, como se puede ver existe un buscador de cedula, y botones que sirven para agregar un nuevo cliente, facturar como consumidor final o eliminar el cliente para asignar uno nuevo a la factura, más abajo se puede observar el buscador de productos con el cual se podrá ir ingresando productos a la factura.

Escogiendo la forma de pago

Una vez la factura esta realizada se elige la forma de pago en una ventana modal con un buscador de las formas de pago existentes.
